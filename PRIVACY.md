# Privacy Policy for YouTube Quick Look

Last updated: May 25, 2026

YouTube Quick Look is a Chrome extension that previews YouTube links and thumbnails directly from the browser.

## Data Collection

YouTube Quick Look does not collect, sell, share, or transmit personal data.

The extension does not use analytics, tracking pixels, advertising SDKs, a remote backend, or third-party telemetry.

## Local Processing

To provide its single purpose, the extension runs locally in the browser and checks hovered page elements for supported YouTube URLs or thumbnail references. This processing happens on the user's device.

The extension may temporarily handle a YouTube video ID in memory so it can open the requested preview. The video ID is not sent to the extension developer.

## Stored Data

The extension stores only these settings in Chrome sync storage:

- Whether the extension is enabled.
- The selected trigger key.

Chrome sync storage is provided by the browser. These settings are used only to restore the user's extension preferences across browser sessions and signed-in Chrome profiles.

## Permissions

The extension uses the `storage` permission to save user preferences.

The extension runs on HTTP and HTTPS pages so it can detect YouTube links and thumbnails wherever the user browses. It does not modify page content except to display the quick-look preview overlay after the user triggers it.

## Third-Party Services

When a user opens a preview, the video is loaded from YouTube in an iframe or player window. YouTube may process requests according to Google's and YouTube's own privacy practices.

## Contact

For privacy questions, contact the publisher through the support contact listed on the Chrome Web Store item.
